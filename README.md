# 📊 Sales Performance Dashboard

## 📌 Project Overview

This project is an interactive Sales Performance Dashboard developed using Microsoft Power BI. It provides business insights through KPI tracking, sales trend analysis, and interactive dashboards.

## 🚀 Dashboard Highlights

- Total Sales KPI
- Total Orders KPI
- Total Quantity KPI
- Average Sales KPI
- Monthly Sales Trend
- Sales by Region
- Sales by Salesperson
- Performance Analysis
- Interactive Slicers

## 🛠️ Tools & Technologies

- Microsoft Power BI
- DAX
- Power Query
- Data Modeling
- Microsoft Excel
